# EXP-MISSION-ToDoApp

## EXP 미션 소개 및 진행 상황
1. ToDo List EXP 미션
   - 기존 ToDo 앱을 확장성을 가지고 개발 진행
   - 새롭게 추가한 내용
     1) 백로그, 체크리스트 본문, 완료된 태스크 총 3가지 항목으로 분리
     2) 다크모드 기능
     3) 검색, 필터링 기능
